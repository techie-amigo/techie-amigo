Html Advanced Table

  • Table tag represents the table format.
  • Caption tag represents tittle of the table for eg: Sales Report
  • colgroup tag is used to define a group of elements.
  • thead tag is used to define table head.
  • tr tag is used to table row.
  • th tag is used to table head.
  • tbody tag is used to write body for the table.
  • td tag is used to write table data in the table.
  • tfoot tag is used to table footer for the table.
  • col is used to specify the columns in the table.
