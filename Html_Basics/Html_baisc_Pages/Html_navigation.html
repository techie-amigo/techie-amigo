    Introduction to HTML

    • Html (Hyper text markup language) is the foundation of web development.
    • It is a markup language that uses tags to define the different elements on web page.
    • Html was created in the early 1990s by "Tim Berners-lee,who also created world wide web.
    • The first version of Html,Html 1.0, was released in 1993.
    • Html is a non- proprietary language,which means that it is not owned by any specific company or organization.
    • It is an open standard that is maintained by the world wide web consortium (W3C).
