Inline & block Level Elements

Inline element

  • In Inline elements the elements takes much space as it can based on the content.
  • If you increase the width or content it will increase only.
  • In Inline elements they do not create a new line.

    • Example:-

    Home About Contact More content...

    Block level element

  • Block level element take up the full weight of the parent container.
  • Block level element creates a new line after themselves and the occupy the full weight of the parent container.
  • Block level element movement strecthes from left to right and it occupies the space available based on the container.

    • Example:-

    Block level element take up the Full weight of the parent container.
