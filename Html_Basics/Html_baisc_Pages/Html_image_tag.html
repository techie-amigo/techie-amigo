Html image tag

  • Image tag is used to embed images into our HTML document.
  • Image tag is also known as self closing tag.

    • We have two ways to use image tags

  • First way is we can download any image and store inside our project and then we can use.
  • Second way is we can copy the url of the image which you want to use in your project and paste it in the project.
