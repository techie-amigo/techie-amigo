Html Text formating tags

  • Text formatting is used to make the text highlight in the website

    • Text format Example

    Strong tag:-
    Strong:- This tag makes the text bold.
    b tag:-
    b:- This tag makes the text bold.
    u tag:-
    u:- This tag is used to underline the text.
    s tag:-
    s:- This tag is used to Strikethrough the text.
    sup tag:-
    sup:- This tag is used to superscript the text.
    sub tag:-
    sub:- This tag is used to subscript the text.
