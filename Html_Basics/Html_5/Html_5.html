HTML 5

  • It is the fifth and current version of HTML, and was first released in October 2014.
  • It improves upon the previous version (HTML4) by introducing new elements,attributes,and APIs.
  • It includes several APIs for bulding web applications, such as wesockets, web workers, and the Geolocation API.

    • HTML4 vs HTML5

  • New semantic elements:: header, nav, article, and footer tag were implemented.
  • Multimedia support:: video and audio tag is implimented.
  • Canvas:: creation of dynamic graphics and animations.
  • Improved forms:: new input types such as search, tel, url, email and attributes.
  • Web storage:: store data locally on the user's device.

    • HTML 5 Sematic

  • what is semantic:: Elements that have a specific meaning and purpose, and convey the structure and organization of the content on a web page.

    • New elements

  • Header:: Represents a container for introductory content aor a set of navigational links.
  • nav:: Represents a sction of a page that contains navigation links.
  • article:: Represents a self-contained composition in a document,such as a blog post or forum post.
  • section:: Represents a thematic grouping of content, such as chapters, headings, or sections of a document.
  • aside:: Represents a section of a page that contains content that is related to the main content, such as sidebar information or pull quotes.
  • footer:: Represents a container for the footer of a document or section.
  • Main:: The "main" element in HTML5 is used to represent the main content of a document.
