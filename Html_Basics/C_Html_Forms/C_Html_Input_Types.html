Html Input Types

  • Text type is which is used to store text in the input place.
  • Password type is which is used to set password and it is store like dots.
  • Checkbox type is which is used to checkbox with tick mark.
  • Radio type is which is used like checkbox.
  • Number type is which is used to store only numbers only.
  • Hidden type is which is used to hide the data.
  • color type is used to choose the color type.
  • Date type is used to specify the date.
  • Datetime-local is used to specify the date-time at one tag.
  • Week type is used to specify the week type.
  • Time type is used to specify the time with pm ,am
  • Month type is used to specify the month type.
  • Email type is used to send an email with this tag.
  • Range type is used to set a range.
  • Search type is used to search bar to search data.
  • File type is used to choose file and select file.
  • Submit type is used to submit the files.
  • Reset type is used to reset the page are data in the form.
