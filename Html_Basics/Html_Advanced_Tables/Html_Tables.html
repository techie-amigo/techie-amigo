Html Tables

  • Tables are used to display data in a tabular format on a web page,and tables can also used
    to organize and present data in a clear and structured way,such as displaying tabular data like financial reports,sports standing or product information.
  • And additionally,tables can also be used for layput papers to create a grid structure of a web page.

    • Row vs Column

  • Horizontal lines is called "Rows". Row starts from left to right.
  • Vertical lines is called "columns" top to bottom.
  • To create Table we are using "Table tag".
  • For table head we are using "th tag".
  • For table row we are using "tr tag".
  • For table data we are using "td tag"
